

# 🎮 Assignment: Games in Python

## 🎯 Objective

Build a classic Hangman game using Python. Practice string manipulation, loops, conditionals, and random selection while creating an interactive word-guessing game.

## 📝 Tasks

### 🛠️ Task 1: Hangman Game Core

#### Description
Create a Hangman game where the player guesses letters to reveal a hidden word before running out of attempts.

#### Requirements
Completed program should:

- Randomly select a word from a predefined list
- Accept letter guesses from the user
- Show current progress (e.g., _ _ _ _)
- Track and display incorrect guesses remaining
- End when the word is guessed or attempts are exhausted
- Display win/lose messages

### 🛠️ Task 2: Game Enhancements (Optional)

#### Description
Add extra features to make your game more fun or challenging.

#### Suggestions
- Allow the player to guess the whole word
- Show letters already guessed
- Add a simple ASCII art hangman
- Let the player play again without restarting the program
